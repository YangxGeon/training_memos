## sar

RHEL 8버전 이후부터는 'dnf명령어 사용.

'top', 'ps'명령어 대신 "sar" 혹은 "sysstat"명령어 기반으로 자원 모니티링 및 확인 권장.

```bash
dnf provides sar
dnf install sysstat -y
systemctl status crond
ls -l /etc/cron.*
crontab -l
systemctl -t timer
rpm -ql sysstat | grep -e .service -e .timer
```
