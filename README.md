## 메모파일 설명

이 디렉터리는 그 동안 진행하였던 강의에 대한 메모를 남겨두는 저장소 입니다.

여기에 보통 남기는 내용들은 온-사이트(On Site Training) 그리고 레드햇에서 진행하였던 강의 입니다.

제가 주기적으로 제거하기 때문에 필요하신분은 그때그때 미리 다운로드를 받으시길 바랍니다.

항상 건강하시길 바랍니다. :)
