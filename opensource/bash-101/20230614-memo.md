## 쉘 변경

```bash
vi /etc/passwd
# bash -> fish

usermod -s /usr/bin/zsh <USERNAME>
~~dnf install chsh -y~~

# util-linux-user
dnf install util-linux-user -y
chsh -l                                     ## 쉘 목록 출력, cat /etc/shells
chsh -s /usr/bin/fish <USERNAME>
```
